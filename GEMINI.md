# SchedulerApp

A scheduling application consisting of a React frontend, a Java backend, and a MySQL database.

## Project Structure

- `frontend/`: React-based user interface.
- `backend/`: Java-based REST API.

## Requirements & Conventions

- **Documentation:** All public methods in both the frontend and backend must include descriptive doc comments (JSDoc for React/TypeScript, Javadoc for Java).
- **Stack:**
    - Frontend: React
    - Backend: Java
    - Database: MySQL
